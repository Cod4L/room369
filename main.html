Room Session Protocol

Core Principles

  1. All participants must know and understand the protocol before the session begins.
  2. No one leaves the room during the session unless it is a group‑approved or emergency exit.
  3. A 20‑minute buffer must pass after the last strike before any exit or window opening.
  4. The veil must remain intact and correctly positioned at all times.
  5. No windows are to be opened during the session unless urgently agreed.
  6. Music volume must remain low enough to hear a knock clearly.
  7. All decisions that affect the group must involve all present parties.

Knock Protocol

  1. Only the coded knock and correct verbal cue are recognized as safe.
  2. If the code is incorrect or missing, complete silence is maintained.
  3. No one approaches or opens the door without proper code.
  4. One designated person handles all door interactions.

One‑Man Exit

  1. Person announces the intention to exit.
  2. Group confirms the room is in proper state.
  3. Door handler listens carefully for movement outside.
  4. If silent, the door is unlocked.
  5. Person exits calmly and quickly.
  6. Door is locked immediately.

Two‑Man Exit

  1. Both individuals prepare along the wall near the door.
  2. First person unlocks the door.
  3. Second person holds the handle.
  4. Both exit calmly but quickly.
  5. Last person locks the door.

Three or More Exit

  1. All individuals align in a straight line along the wall at a 90° angle to the door.
  2. First person unlocks the door.
  3. Second person holds the handle.
  4. The line exits smoothly and quickly.
  5. Last person locks the door.

Emergency Single Exit

  1. Person declares an emergency exit.
  2. Group secures the room and confirms the veil is intact.
  3. Door handler listens for movement outside.
  4. If silent, the door opens briefly.
  5. Person exits immediately.
  6. Door locks at once.
  7. Remaining group resets the 20‑minute buffer.
